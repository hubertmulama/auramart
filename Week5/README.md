# AuraMart 🛍️
### Smart E-Commerce Web Application
**BIT3208 — Advanced Web Design and Development**

## About
AuraMart is a full-stack e-commerce platform built as a capstone project for BIT3208.
It allows users to browse products, manage a shopping cart, place orders, and administrators
to manage products, orders, and users.

## Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Server:** XAMPP (Apache)

## Features
- User registration and login with secure password hashing
- Role-based access (Admin / Customer)
- Product catalog with search and category filter
- Shopping cart with localStorage persistence
- Checkout and order placement
- Admin panel — manage products, orders, and users
- Customer dashboard with order history
